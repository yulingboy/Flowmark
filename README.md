# Flowmark

简洁高效的浏览器起始页，支持快捷方式管理、插件系统、个性化设置等功能。

## ✨ 功能特性

- 🔗 **快捷方式管理** - 自定义网站快捷入口，支持拖拽排序和分组
- 🔍 **多引擎搜索** - 支持 Google、Bing、百度等多种搜索引擎切换
- 🎨 **壁纸设置** - 支持自定义背景图片和纯色背景
- 🕐 **时钟显示** - 可配置的时间日期显示
- 🧩 **插件系统** - 内置万年历等实用插件，支持多种尺寸
- 💾 **数据管理** - 支持数据导入导出和一键还原
- ⌨️ **快捷键** - 支持键盘快捷操作

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 构建浏览器扩展
pnpm build:ext
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 聚焦搜索框 |
| `Ctrl + N` | 添加新快捷方式 |
| `Ctrl + ,` | 打开设置 |
| `Esc` | 关闭弹窗 |

## 🛠️ 技术栈

- React 19 + TypeScript
- Vite 5
- Tailwind CSS 4
- Zustand 5
- Ant Design 6
- @dnd-kit

## 📦 安装为浏览器扩展

1. 运行 `pnpm build:ext` 构建扩展
2. 打开浏览器扩展管理页面
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `dist` 目录
